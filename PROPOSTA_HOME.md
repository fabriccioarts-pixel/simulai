# Proposta de Melhorias Simulai Home 🚀

Para transformar o Simulai em um ambiente de alto nível para concurseiros, as melhorias devem focar em **Engajamento**, **Frequência** e **Personalização**.

## 1. Experiência Multi-Fileiras (Implementado ✅)
A home não deve ser apenas uma lista simples. Agora o conteúdo é agrupado por:
- **Novidades:** Simulados recém-carregados.
- **Por Matéria:** Direito Administrativo, Economia, Contabilidade, etc.
- **Em Breve:** Para gerar antecipação.

## 2. Continuar Estudando (Recurso Prioritário)
Um bloco exclusivo no topo (abaixo do Hero) que mostra:
- O último simulado que o usuário não terminou.
- Barra de progresso visual (Ex: "Faltam 4 questões para acabar").
- Botão "Continuar de Onde Parei".

## 3. Desafio Diário (Viralização)
Um card de destaque no banner (Hero) com a **Questão do Dia**:
- Uma questão aleatória do banco de dados que aparece para todos.
- Rank diário: Quem acertou em menos tempo?
- Isso força o usuário a abrir o app pelo menos 1x por dia.

## 4. Gamificação (EAD Premium)
- **Racha de Estudos:** Mostrar quantos simulados o usuário fez seguidos.
- **Nível de Proficiência:** Gráfico de teia no perfil mostrando qual matéria ele é melhor (Baseado nos acertos).
- **Simulado Simulado:** Definir uma data (ex: Todo domingo às 14h) para um simulado "Ao Vivo" com ranking nacional.

## 5. Hub de Materiais Complementares
Abaixo da fileira de simulados, adicionar fileiras de **Anexos e PDFs**:
- Material de apoio que você cadastrar no Admin.
- Resumos estratégicos (Lei seca, esquemas).

## 6. Personalização Visual (Modo Dark/Glass)
- **Capa Personalizável:** O Admin agora pode subir URLs de imagens, use pôsteres vibrantes para as matérias.
- **Efeito de Foco na Hero:** O fundo da hero pode ser o pôster do simulado mais recente.
